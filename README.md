🖥️ **Linux Command #2: `pwd`**  

💡 **What it does**: Shows your current directory.  

**Try it!** Open your terminal and type:  

```bash
pwd
```  
$  Output: /home/your-user-name  

🔹 **Pro Tip**: Combine with `ls` to see where you are & what’s there!
